# Zia Supplement House

## Project Overview
Zia Supplement House is designed to provide a comprehensive solution for managing dietary supplements. The project aims to help users track their supplement intake, understand nutritional values, and manage their health better.

## Setup Instructions

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/scorpio28050-sys/zia-supplement-house.git
   ```

2. **Navigate into the project directory:**  
   ```bash
   cd zia-supplement-house
   ```

3. **Install dependencies:**  
   ```bash
   npm install
   ```

4. **Run the application:**  
   ```bash
   npm start
   ```

5. **Access the application:**  
   Open your web browser and go to `http://localhost:3000` (or the designated port).
